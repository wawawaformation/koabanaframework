Detailed Description

SessionBag : gestion d'un sous-ensemble de données de session.

Permet d'accéder, modifier, supprimer et lister des valeurs dans un tableau de session, typiquement utilisé pour isoler des groupes de clés (ex : flash, user, panier).

Les opérations sont faites par référence sur le tableau fourni au constructeur,

Returns
void ce qui garantit la synchronisation avec la session PHP native ou tout autre stockage.

Exemple d'utilisation : $bag = new SessionBag($_SESSION['user']); $bag->set('id', 42); $id = $bag->get('id');
